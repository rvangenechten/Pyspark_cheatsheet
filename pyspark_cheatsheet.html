Z-Order Optimization
OPTIMIZE + ZORDER
spark.sql(""" OPTIMIZE delta.`/path/to/table` ZORDER BY (country, event_date) """)
✓ When to use
High-cardinality cols in WHERE/JOIN. 2–4 cols max. Co-locates data → fewer files scanned.
⚠ Avoid on
Partition cols, boolean/low-cardinality cols, cols never used in filters.
VACUUM — Storage Cleanup
Run VACUUM
# Dry run first spark.sql("VACUUM delta.`/path` DRY RUN") # Default = 168 h (7 days) spark.sql("VACUUM delta.`/path/to/table`") # Custom retention spark.sql("VACUUM delta.`/path` RETAIN 168 HOURS")
⚠ Caution
Never below 168 h — breaks time travel. Never below streaming checkpoint lag.
Delta Lake — Write, Merge & Time Travel
Write modes
df.write.format("delta") \ .mode("overwrite").save(path) # overwrite|append|ignore|error .partitionBy("date").save(path)
Merge / Upsert
from delta.tables import DeltaTable tgt = DeltaTable.forPath(spark, path) tgt.alias("t").merge( src.alias("s"), "t.id = s.id") \ .whenMatchedUpdateAll() \ .whenNotMatchedInsertAll() \ .execute()
Time Travel
# by version spark.read.format("delta") \ .option("versionAsOf", 3) \ .load(path) # by timestamp .option("timestampAsOf","2024-01-01") tgt.restoreToVersion(2)
History & Schema
